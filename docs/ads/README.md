# KJV Words Collection — Ad & Marketing Copy

Ready-to-use text for Facebook, Instagram, Threads, Blogger, Naver Blog, Tistory, WordPress, newsletter, and email. Replace `[YOUR_URL]` with your live site URL before publishing.

**Character limits (approximate):**
- Facebook ad primary text: ~125 chars for best engagement; link posts can be longer.
- Instagram caption: 2,200 max; first ~150 chars show before “more”.
- Threads: 500 chars.
- Meta (FB/IG) headline: 40 chars, description 30 chars.

- `instagram.md`
- `threads.md`
